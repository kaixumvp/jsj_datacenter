# 实时监测数据列表 接口文档

## 基本信息

表格

| 项目   | 内容                                       |
| ---- | ---------------------------------------- |
| 接口名称 | 实时监测数据列表                                 |
| 请求地址 | {ip}:{port}/prod-api/environmentNew/list |
| 请求方式 | `GET`                                    |
| 访问权限 | 匿名访问（无需登录）                               |
| 接口描述 | 获取环境监测实时数据集合，包含环境指标、设备信息、气象、定位等数据        |

## 请求参数

无请求参数

## 响应数据

### 整体返回结构

json

```
{
  "code": 200,
  "msg": "操作成功",
  "data": [
    // 数组元素为单条监测数据对象
  ]
}
```

### 字段说明

#### 外层公共返回

表格

| 字段名  | 类型     | 说明             |
| ---- | ------ | -------------- |
| code | int    | 响应状态码，200 = 成功 |
| msg  | string | 响应提示信息         |
| data | array  | 业务数据，数组格式      |

#### 数据项 `EnvironmentMonitorNewVo`

继承 `EnvironmentMonitorNew` 实体，字段明细如下：

表格

| 字段名               | 数据类型          | 字段说明         | 单位 / 补充备注      |
| ----------------- | ------------- | ------------ | -------------- |
| id                | Long          | 主键 id        | 自增主键           |
| deviceId          | Integer       | 设备地址 / 设备 ID | 关联设备编号         |
| node              | Integer       | 节点           | 设备所属节点编号       |
| pm2               | Float         | pm2.5        | ug/m³          |
| pm10              | Float         | pm10         | ug/m³          |
| tsp               | Float         | 总悬浮颗粒物       | ug/m³          |
| noise             | Float         | 噪声           | dB (分贝)        |
| temperature       | Float         | 温度           | -              |
| humidity          | Float         | 湿度           | -              |
| windPower         | Float         | 风力           | m/s            |
| windSpeed         | Float         | 风速           | m/s            |
| windDirect        | Float         | 风向 (方位数值)    | 原始数值           |
| windDirectDegrees | Float         | 风向 (角度)      | 风向度数           |
| lng               | Float         | 经度           | 坐标经度           |
| lat               | Float         | 纬度           | 坐标纬度           |
| coordinateType    | Integer       | 坐标类型         | 枚举见下方          |
| relayStatus       | String        | 继电器状态        | 设备继电器运行状态      |
| createTime        | LocalDateTime | 创建时间         | 数据采集上报时间       |
| deviceName        | String        | 设备名称         | VO 扩展字段        |
| position          | String        | 设备位置         | VO 扩展字段，设备安装地点 |
| windDir           | String        | 风向           | VO 扩展字段，中文方位描述 |

##### coordinateType 枚举说明

表格

| 值   | 含义                                  |
| --- | ----------------------------------- |
| 0   | 百度经纬度                               |
| 1   | 中国联通基站定位（lat=CELLID，lng=LAC，MNC=01） |
| 2   | 中国移动基站定位（lat=CELLID，lng=LAC，MNC=00） |
| 3   | GPS 经纬度                             |
